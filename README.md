# omniwheel-rc

A remote control with Joysticks for omniwheel vehicles built with Vue.js.

## Setup

### Installation

```bash
npm install
```

### Compiles and hot-reloads for development

Localhost:

```bash
npm run dev
```

Network:

```bash
npm run dev -- --host
```

### Compiles and minifies for production

```bash
npm run build
```
